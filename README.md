# Car2Fix Website

Professional automotive website for Car2Fix mechanical and body shop services in Newark and Linden, NJ.

## Tech Stack

- **React 18** - UI Framework
- **Vite** - Build Tool
- **Tailwind CSS** - Styling
- **Framer Motion** - Animations
- **React Router** - Navigation
- **Lucide React** - Icons

## Features

- 🌓 Dark/Light mode toggle
- 📱 Fully responsive design
- ✨ Smooth animations
- 📝 Contact forms on service pages
- 🗺️ Google Maps integration
- 🎨 Bento grid design aesthetic

## Pages

- **Home** (`/`) - Landing page with hero, locations, features, and testimonials
- **Mechanical Shop** (`/mech-shop`) - Services and contact for Newark location
- **Body Shop** (`/body-shop`) - Services and contact for Linden location
- **About** (`/about`) - Company story and values

## Getting Started

### Prerequisites

- Node.js 18+ installed
- npm or yarn

### Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment

This site is configured for Vercel deployment. Simply connect your GitHub repository to Vercel and it will auto-deploy.
